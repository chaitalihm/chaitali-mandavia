Model assumptions
Manual baseline
Volume times minutes per task divided by 60, times hourly wage. What you are currently spending with people doing this work.
Model API fees
These numbers are estimated based on mid-tier LLM pricing as a reference point. Current Anthropic pricing is $3.00/$15.00 per million tokens for Claude Sonnet 4.6. Actual costs depend on the model you choose and change frequently, so check current pricing before using these numbers for real decisions.
Token counts by use case
Document extraction assumes around 45K input tokens per task, support routing around 4K, and RAG around 15K. These are estimates based on typical document lengths for each category.
Human oversight (fallback rate)
The percentage of tasks that still need a person to review or correct the output. Document extraction assumes 15%, support routing 8%, RAG 5%. These are estimated based on typical task complexity.
Upfront implementation cost
Covers setup, integration work, and prompt engineering. Estimated at a base of $25K-$40K depending on use case, plus $0.50 per unit of monthly volume. This is a rough range based on typical enterprise AI projects.
What is not included
Data licensing fees, change management, training time, and ongoing monitoring are not modeled. These vary too much by organization but can add 20-40% to the real number.
